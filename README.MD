# Bukkit Example Plugin

### WARNING!
##### 此插件模板仅适用于Minecraft : Java Edition 1.7 之前的Bukkit
##### 此处引用的库是 Beta 1.8.1 的 CraftBukkit Server

## build

### 使用Intellij IDEA 打开项目，点击左上角的横线。
### 再点击Build >> build artifacts, 
### 在构建工件(build artifacts)选项中选择 build。
### 插件的Jar就构建好了

###

### 如果你还想更改插件名字
### 可以去file >> Project Structure >> artifacts
### 进行更改
